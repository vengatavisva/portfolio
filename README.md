# Vengata Visva Saravanan - Professional Portfolio

Welcome to my personal 3D interactive portfolio, showcasing my projects, skills, and professional journey.

## 🚀 Features

- **Interactive 3D Elements:** Built using `@react-three/fiber` and `@react-three/drei` for engaging user experiences.
- **Smooth Animations:** Powered by `framer-motion` for fluid page transitions and interactive components.
- **Modern UI:** Styled with Tailwind CSS for a fully responsive, beautiful, and sleek design.
- **Performance Optimized:** Fast bundling and hot module replacement with Vite.

## 🛠️ Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **3D Graphics:** Three.js / React Three Fiber
- **Animations:** Framer Motion

## ⚙️ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v16+ recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-github-repo-url>
   ```

2. Navigate into the directory:
   ```bash
   cd portfolio
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

The application will be running at `http://localhost:5173`.

## 📦 Build for Production

To create a production-ready build, run:
```bash
npm run build
```

You can then preview the build with:
```bash
npm run preview
```
